# MongoDB Assignment – collegeDB

## Student Database Operations using MongoDB

This repository contains MongoDB shell commands for the following tasks:

1. Creating a database named `collegeDB`
2. Creating a collection named `students`
3. Inserting documents into the collection
4. Retrieving documents
5. Updating a document
6. Deleting a document

## Files Included
- `mongodb_commands.js` – MongoDB shell commands for all questions

## Tools Used
- MongoDB
- mongosh
